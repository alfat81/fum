# Официальный сайт комиссии по фигурному управлению мотоциклом (мотоджимхана) МФР

## Описание
Этот репозиторий содержит исходный код официального сайта комиссии по фигурному управлению мотоциклом при Мотоциклетной Федерации России. Сайт разработан для информирования о деятельности комиссии, публикации календаря соревнований и предоставления доступа к нормативным документам.

## Основные разделы сайта:
- **Главная страница** - общая информация о комиссии
- **О комиссии** - история создания, миссия и цели
- **Структура комиссии** - руководство и региональные представители
- **Календарь соревнований** - официальные соревнования по фигурному управлению мотоциклом
- **Полезные документы** - правила, регламенты и другие нормативные документы
- **Контакты** - контактная информация комиссии
- **Админ-панель** - управление содержимым сайта

## Особенности:
- Полностью статический сайт, размещаемый на GitHub Pages
- Админ-панель для управления контентом без серверной части
- Все документы хранятся в Яндекс.Диске по адресу: https://disk.yandex.ru/d/ubxzU_0RzkFsJw
- Данные сохраняются в localStorage браузера
- Адаптивный дизайн для всех устройств
- Простая и понятная навигация

## Технологии:
- HTML5
- CSS3
- JavaScript
- GitHub Pages для хостинга

## Установка и запуск:
1. Склонируйте репозиторий: `git clone https://github.com/alfat81/fum.git`
2. Откройте любой HTML-файл в браузере для просмотра сайта
3. Для размещения на GitHub Pages:
   - Создайте репозиторий на GitHub
   - Загрузите все файлы в репозиторий
   - В настройках репозитория включите GitHub Pages (ветка main, папка root)
