# Сайт комиссии по фигурному управлению мотоциклом (мотоджимхане)

Официальный сайт комиссии по фигурному управлению мотоциклом при Мотоциклетной Федерации России.

## Основные разделы:

- **Главная страница** - общая информация и навигация
- **О комиссии** - история создания, миссия и цели
- **Структура комиссии** - руководство и региональные представители
- **Календарь соревнований** - официальные соревнования по фигурному управлению мотоциклом
- **Полезные документы** - правила, регламенты и нормативные документы
- **Контакты** - контактная информация комиссии
- **Админ-панель** - управление контентом сайта

## Особенности:

- Полностью автономный сайт без использования внешних API
- Админ-панель для управления контентом с защитой по паролю
- Все документы хранятся в Яндекс.Диске по адресу: https://disk.yandex.ru/d/ubxzU_0RzkFsJw
- Адаптивный дизайн для всех устройств
- Темно-синий фирменный стиль с логотипами МФР и Министерства спорта РФ в футере
- Интерактивный календарь соревнований
- Пошаговая история развития комиссии
- Функционал редактирования региональных представителей через локальное хранилище

## Установка:

1. Склонируйте репозиторий
2. Загрузите все файлы на GitHub Pages
3. Разместите логотипы в папке `images/`:
   - `logo.png` - логотип комиссии
   - `mfr-logo.png` - логотип МФР
   - `minsport-logo.png` - логотип Министерства спорта РФ
4. Измените пароль админ-панели в файле `script.js` (переменная `correctPassword`)
